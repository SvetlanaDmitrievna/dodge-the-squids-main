# Dodge the Creeps — игра на выживание

Это учебный проект, созданный в рамках учебной практики.

🎮 Инструкция по запуску игры
Способ 1. Запуск через исполняемый файл (.exe) — для пользователя
Этот способ подходит для тех, у кого не установлен Godot.

Скачайте архив с игрой с GitHub ( https://github.com/SvetlanaDmitrievna/dodge-the-squids-main/tree/main/game%20final)

Распакуйте архив в любую папку на компьютере.

Откройте папку с игрой. В ней должны находиться два файла:

.exe — исполняемый файл

.pck — файл с ресурсами (картинки, звуки)

Запустите файл .exe (двойным щелчком мыши).

Игра начнётся. Управление осуществляется клавишами-стрелками (↑ ↓ ← →).

Важно: Файлы .exe и .pck должны находиться в одной папке, иначе игра не запустится.

Способ 2. Запуск через Godot Engine — для разработчика / преподавателя
Этот способ подходит для тех, у кого установлен Godot, и кто хочет посмотреть исходный код.

Что потребуется:
Установленный Godot Engine 4.6.1 (обычная версия, не .NET)

Скачанный или склонированный репозиторий с игрой

Шаги для запуска:
Скачайте или клонируйте репозиторий с GitHub:

Ссылка на репозиторий: https://github.com/SvetlanaDmitrievna/dodge-the-squids-main

Запустите Godot Engine.

Откроется менеджер проектов (окно со списком проектов).

Импортируйте проект:

Нажмите кнопку «Импортировать» (Import).

В открывшемся окне найдите папку, куда вы скачали проект, и выберите файл project.godot.

Нажмите «Импортировать и редактировать» (Import & Edit).

Запустите игру:

В редакторе Godot нажмите клавишу F5 (или кнопку «Запуск проекта» в правом верхнем углу).

Управление:

Клавиши-стрелки (↑ ↓ ← →) — движение персонажа.

В главном меню выберите уровень сложности (Easy / Normal / Hard) перед нажатием Start.

🧪 Проверка работоспособности
После запуска игры:

Должно появиться окно с игровым полем.

В правом верхнем углу должен быть выпадающий список выбора сложности.

Персонаж должен двигаться при нажатии на стрелки.

При столкновении с врагом игра должна завершаться и показывать сообщение «Game Over». 

##  Уровни сложности

В игру добавлены три уровня сложности, которые влияют на скорость врагов и частоту их появления:

| Сложность | Скорость врагов | Интервал появления |
|-----------|-----------------|-------------------|
| Easy | 100-200 | 1.2 сек |
| Normal | 150-250 | 0.8 сек |
| Hard | 500-550 | 0.2 сек |

Выбор сложности осуществляется в выпадающем списке перед началом игры.
